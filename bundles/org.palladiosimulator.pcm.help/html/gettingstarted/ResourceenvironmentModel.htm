Creating a Resource Environment Model

  1. In the Package Explorer view, select the project. Then right-click on the project and select New > Other... from the project´s context menu.
  2. Select Example EMF Model Creation Wizards > Resource Environment Model then click Next to start the wizard.
  3. On the next page click again Next.
  4. Select the Model Object Resource Environment and click Finish.
  5. Right-click in the resource environment view and select Load Resource...
  6. In the Load Resource wizard select Browse Workspace...
  7. Select your project and your resource type. Then click Ok.
  8. Click again Ok in the Load Resource wizard.
