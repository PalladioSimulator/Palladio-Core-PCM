Creating a Allocation Model

  1. In the Package Explorer view, select the project. Then right-click on the project and select New > Other... from the project´s context menu.
  2. Select Palladio Modeling > PCM Allocation Diagram then click Next to start the New allocation Diagram wizard.
  3. On the next page click Next and again Next.
  4. Then you get the response "No resource environment selected". Click on Change to select the resource environment.
  5. In the Load Resource Wizard click on Browse Workspace... and select your project and your resource environment. Then click Ok.
  6. Click again Ok in the Load Resource wizard.
  7. In the New allocation Diagram wizard click Next.
  8. And you get the response "No system selected". Click on Change to select the system. Then click Ok.
  9. Click again Ok in the Load Resource wizard.
  10. In the New allocation Diagram wizard click Finish.
