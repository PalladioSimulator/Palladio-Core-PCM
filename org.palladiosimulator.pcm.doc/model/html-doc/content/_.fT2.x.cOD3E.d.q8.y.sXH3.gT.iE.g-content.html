Zuordnung <Unnamed Association>

To express the performance-relevant resource interaction of the modelled computations, an InternalAction contains a set of ParametricResourceDemands, each directed at a specific ProcessingResourceType. An InternalAction may have at most one resource demand for each ProcessingResourceType. Analysis or simulation tools sequentialise the set of demands and put load on the referenced resources one after another.
